## ▶️ Running the Projects

### Requirements

* A compatible .NET SDK
* Visual Studio, JetBrains Rider, Visual Studio Code, or another C# editor

### Clone the repository

```bash
git clone https://github.com/rkeppler42/csharp-players-guide-solutions.git
cd csharp-players-guide-solutions
```

### Build the solution

```bash
dotnet build
```

### Run a specific challenge

Navigate to the challenge directory and run:

```bash
dotnet run
```

Alternatively, provide the project path from the repository root:

```bash
dotnet run --project path/to/ChallengeName.csproj
```
